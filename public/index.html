🎬 YuJi API

Social Media Content Parser for Instagram, X (Twitter) & YouTube

Supported Platforms

📷 Instagram 🐦 X (Twitter) 📺 YouTube

API Endpoint

GET /api/parse?url={encoded_url}
Note: Make sure to URL-encode your social media link when passing it as the url parameter.

Features

  • Parse Instagram posts, reels, and stories
  • Extract X (Twitter) tweets with media
  • Get YouTube video information and thumbnails
  • No authentication required
  • CORS-enabled for cross-origin requests
  • Fast serverless processing

Example Request

JavaScript Fetch

const url = 'https://www.instagram.com/p/example/';
const apiUrl = `/api/parse?url=${encodeURIComponent(url)}`;

fetch(apiUrl)
  .then(res => res.json())
  .then(data => {
    if (data.code === 0) {
      console.log(data.data);
      // { platform, title, author, imageList[], videoUrl }
    }
  });

Response Format

{
  "code": 0,
  "msg": "ok",
  "data": {
    "platform": "instagram",
    "title": "Post title",
    "author": "username",
    "imageList": ["url1", "url2"],
    "videoUrl": "video_url"
  }
}
